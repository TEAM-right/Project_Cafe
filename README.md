# project-cafe
임의의 사업을 선택하고 그 사업에 필요한 프로그램을 제작한다.<br>
카페 업종으로 주제 선정.

## 구현한 기능
* 로그인
* 메뉴 선택 화면
* 직원 관리
* 판매 등록
* 적립 화면
* 재고 관리
* 매출 조회
